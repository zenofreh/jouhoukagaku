# 情報科学基礎 クイズ

画像から作成した情報科学基礎のクイズアプリです。

## 使い方

`index.html` をブラウザで開くと利用できます。

- ランダム版: 各問題番号から1問ずつ出題
- 全問版: 収録されている全45問を出題
- 回答すると自動で次の問題へ進みます
- 途中で「ホームへ戻る」から中断できます
- 最後に成績と回答確認を表示します

## ファイル

- `index.html`: 画面
- `styles.css`: デザイン
- `app.js`: クイズの進行ロジック
- `data/questions.js`: アプリが読み込む問題データ
- `data/questions.json`: 編集用の問題データ
